1) Fundamentos

Los sitios estáticos ofrecen HTML ya renderizado, lo que es ideal para SEO. La clave está en una buena estructura, metadatos correctos, contenido accesible y rendimiento sólido.

Regla de oro: cada página debe responder claramente a una intención de búsqueda.

2) Metadatos esenciales

  • Title: ~50–60 caracteres, único y descriptivo.
  • Meta description: ~150–160 caracteres, persuasiva.
  • Viewport, charset y lang en HTML.
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>SEO técnico para sitios estáticos | Tu Marca</title>
  <meta name="description" content="Guía práctica para optimizar títulos, metadatos, estructura y datos enriqueci... ">
</head>

3) Semántica y encabezados

Usa un solo <h1> por página y estructura el contenido con h2, h3, listas y elementos semánticos (<main>, <nav>, <article>, <footer>).

<main>
  <h1>SEO técnico para sitios estáticos</h1>
  <section><h2>Metadatos</h2>...</section>
  <section><h2>Estructura</h2>...</section>
</main>

4) URLs, canonicals y estructura

  • Prefiere URLs limpias: /blog/seo-estatico.
  • Usa canonical para evitar duplicados.
  • Mantén consistencia con o sin slash final.
<link rel="canonical" href="https://tusitio.com/blog/seo-estatico">
<!-- Si publicas la misma página en /blog/seo-estatico/ y /blog/seo-estatico -->

5) sitemap.xml y robots.txt

Incluye todas las URLs indexables en sitemap.xml y referencia el sitemap desde robots.txt.

<!-- sitemap.xml -->
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://tusitio.com/</loc>
    <lastmod>2025-05-20</lastmod>
  </url>
  <url>
    <loc>https://tusitio.com/blog/seo-estatico</loc>
    <lastmod>2025-05-22</lastmod>
  </url>
</urlset>

# robots.txt
User-agent: *
Allow: /

Sitemap: https://tusitio.com/sitemap.xml

6) Open Graph y Twitter

Mejora la apariencia al compartir. Para sitios estáticos, define estas metas en cada página.

<!-- Open Graph -->
<meta property="og:type" content="article">
<meta property="og:title" content="SEO técnico para sitios estáticos">
<meta property="og:description" content="Guía práctica de metadatos, estructura, datos enriquecidos y rendimiento.">
<meta property="og:url" content="https://tusitio.com/blog/seo-estatico">
<meta property="og:image" content="https://tusitio.com/og/seo-estatico.jpg">

<!-- Twitter -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="SEO técnico para sitios estáticos">
<meta name="twitter:description" content="Checklist accionable para sitios estáticos.">
<meta name="twitter:image" content="https://tusitio.com/og/seo-estatico.jpg">

7) Datos estructurados (JSON-LD)

Usa JSON-LD para ayudar a los motores de búsqueda a entender tu contenido.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "SEO técnico para sitios estáticos",
  "datePublished": "2025-05-22",
  "author": {"@type":"Person","name":"Pablo Rodriguez"},
  "publisher": {"@type":"Organization","name":"Binary Pixis"},
  "mainEntityOfPage": "https://tusitio.com/blog/seo-estatico",
  "image": ["https://tusitio.com/og/seo-estatico.jpg"]
}
</script>
<script type="application/ld+json">
{
  "@context":"https://schema.org",
  "@type":"BreadcrumbList",
  "itemListElement":[
    {"@type":"ListItem","position":1,"name":"Blog","item":"https://tusitio.com/blog"},
    {"@type":"ListItem","position":2,"name":"SEO","item":"https://tusitio.com/blog/seo"},
    {"@type":"ListItem","position":3,"name":"SEO estático","item":"https://tusitio.com/blog/seo/seo-estatico"}
  ]
}
</script>

8) Imágenes y ALT

  • Nombres descriptivos: seo-estatico-hero.jpg.
  • alt significativo (o vacío si decorativa).
  • Dimensiones fijas para evitar CLS.

9) Internacionalización (hreflang)

Si tienes versiones por idioma/región, declara alternates y canónica.

<link rel="canonical" href="https://tusitio.com/es/seo-estatico">
<link rel="alternate" href="https://tusitio.com/es/seo-estatico" hreflang="es">
<link rel="alternate" href="https://tusitio.com/en/static-seo" hreflang="en">
<link rel="alternate" href="https://tusitio.com/" hreflang="x-default">

11) Performance & Core Web Vitals

El rendimiento impacta en SEO: mejora LCP (servidor rápido y HTML liviano), evita CLS (espacios reservados) y cuida INP (menos JS bloqueante).

<!-- Reservar espacio para imágenes y fuentes con width/height y font-display: swap -->
@font-face{ font-family:Inter; src:url("/fonts/Inter-400.woff2") format("woff2"); font-display: swap; }

Checklist rápida

  • Título único (50–60) + meta description (150–160)
  • Estructura semántica con un solo h1
  • URLs limpias y consistentes + rel="canonical"
  • sitemap.xml y robots.txt publicados
  • Open Graph/Twitter definidos por página
  • JSON-LD: Article, Breadcrumb, Organization
  • Imágenes con nombre descriptivo, alt y dimensiones
  • hreflang para idiomas/regiones
  • Enlazado interno y breadcrumbs visibles
  • LCP/INP/CLS medidos y optimizados
